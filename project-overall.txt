RxSmart Project Overall (Master Checklist)

Goal:
สร้างเครื่องกายภาพบำบัดที่วัดการเคลื่อนไหวจากเซนเซอร์ 8 จุด
(ต้นแขนซ้าย/ขวา, ปลายแขนซ้าย/ขวา, ต้นขาซ้าย/ขวา, ปลายขาซ้าย/ขวา)
และสรุปผลเป็นมุม (angle), ความเร็ว (speed), ท่าทาง (posture), จำนวนรอบ (rep)

==================================================
1) Product Definition
==================================================
[ ] นิยาม use case หลักตามโรค/อาการที่ต้องรองรับ
[ ] กำหนดท่ากายภาพบำบัดหลักที่ระบบต้องวัด
[ ] นิยาม KPI ของระบบ (ความแม่นยำ, latency, ความเสถียร)
[ ] ระบุเกณฑ์ผ่าน/ไม่ผ่านของแต่ละท่า

==================================================
2) Hardware & Sensor Layer
==================================================
[ ] เลือก/ยืนยันชนิดเซนเซอร์ที่ใช้ทั้ง 8 จุด
[ ] ออกแบบการติดตั้งเซนเซอร์บนร่างกายให้ซ้ำได้จริง
[x] อ่านข้อมูลเซนเซอร์จริงครบ 8 ตัวพร้อม timestamp
[x] ทำ calibration flow ก่อนเริ่ม session
[ ] เพิ่มการกรองสัญญาณเพื่อลด noise/drift
[ ] จัดการ battery/power และความปลอดภัยอุปกรณ์

==================================================
3) Motion Algorithm Layer
==================================================
[x] สร้าง data model สำหรับข้อมูล 8 เซนเซอร์
[x] คำนวณมุมที่จำเป็นต่อแต่ละท่าฝึก
[x] คำนวณความเร็วการเคลื่อนไหว
[x] ทำระบบนับจำนวนรอบ (rep counter)
[x] วิเคราะห์ posture ว่าถูก/ผิดท่า
[x] ทำกติกาเตือนท่าเสี่ยงบาดเจ็บ

==================================================
4) Embedded Firmware (ESP32)
==================================================
[x] Wi-Fi provisioning (Captive Portal)
[x] บันทึกค่า Wi-Fi และ recovery flow
[x] Internet watchdog
[x] Device registration + command polling + telemetry พื้นฐาน
[x] telemetry v2: ส่งข้อมูลแยก 8 เซนเซอร์ + angle/speed/posture/rep
[x] state machine session: idle -> calibrate -> exercise -> complete
[ ] fault handling (sensor missing, packet loss, reconnect)

==================================================
5) Cloud/API/Data
==================================================
[x] ออกแบบ API contract สำหรับ session กายภาพบำบัด
[x] ออกแบบฐานข้อมูล (user/session/exercise/metrics/events)
[x] จัดเก็บข้อมูล time-series สำหรับการวิเคราะห์ย้อนหลัง
[ ] ทำ auth และสิทธิ์เข้าถึงข้อมูลผู้ป่วย/ผู้ดูแล
[ ] เพิ่ม logging/monitoring สำหรับอุปกรณ์และ API

==================================================
6) App / Dashboard
==================================================
[x] หน้าติดตามแบบ real-time (angle/speed/posture/rep)
[ ] สรุปผลราย session พร้อมคะแนนคุณภาพท่า
[ ] ประวัติแนวโน้มอาการรายสัปดาห์/รายเดือน
[x] ระบบแจ้งเตือนเมื่อทำท่าผิดหรือเกินช่วงปลอดภัย
[ ] หน้าจัดการโปรแกรมฝึกตามโรค/อาการ

==================================================
7) Clinical / Safety / Compliance
==================================================
[ ] ทบทวนเกณฑ์ทางกายภาพบำบัดร่วมกับผู้เชี่ยวชาญ
[ ] วางขอบเขตการใช้งานและข้อจำกัดของระบบ
[ ] จัดทำคำแนะนำการติดตั้งและใช้งานอย่างปลอดภัย
[ ] นโยบายความเป็นส่วนตัวและการคุ้มครองข้อมูลสุขภาพ

==================================================
8) Testing & Validation
==================================================
[ ] unit test สำหรับ algorithm หลัก
[ ] integration test (sensor -> firmware -> cloud -> dashboard)
[ ] test ภาคสนามกับผู้ใช้งานจริงหลายกลุ่ม
[ ] ตรวจสอบความแม่นยำเทียบมาตรฐานอ้างอิง
[ ] ทดสอบความทนทานระยะยาว (long run stability)

==================================================
9) Deployment & Operations
==================================================
[ ] เวิร์กโฟลว์ release firmware และ rollback
[ ] คู่มือติดตั้งหน้างานและ troubleshooting
[ ] ระบบ support/incident response
[ ] แผนบำรุงรักษาและอัปเดตโปรแกรมฝึก

[ ] M2: คำนวณ angle/speed/rep ได้เสถียร
[x] M3: posture detection + alert ใช้งานได้จริง
