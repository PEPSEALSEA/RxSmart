================================================================================
  RxSmart — Hardware Wiring Guide (Production Setup)
  TCA9548A I2C Multiplexer × 1  +  MPU6050 (GY-521) × 8
  ESP32 DevKit V1 (38-pin, DOIT)
================================================================================

────────────────────────────────────────────────────────────────────────────────
ภาพรวม Board Structure
────────────────────────────────────────────────────────────────────────────────

  ESP32 DevKit V1
  ┌─────────────────────────────┐
  │  GPIO21 (SDA) ──────────────┼──► TCA9548A SDA  ◄── I2C Master bus
  │  GPIO22 (SCL) ──────────────┼──► TCA9548A SCL  ◄── I2C Master bus
  │  3V3 ────────────────────── ┼──► TCA9548A VCC
  │  GND ────────────────────── ┼──► TCA9548A GND
  │                             │    TCA9548A A0 → GND  ┐
  │                             │    TCA9548A A1 → GND  ├─ I2C addr = 0x70
  │                             │    TCA9548A A2 → GND  ┘
  │                             │    TCA9548A RESET → 3V3 (keep HIGH)
  └─────────────────────────────┘

  TCA9548A
  ┌──────────────────────────────────────────────────────────────────────────┐
  │ SD0/SC0 ──► MPU6050 #0  SDA/SCL  [CH0]  →  left_upper_arm              │
  │ SD1/SC1 ──► MPU6050 #1  SDA/SCL  [CH1]  →  right_upper_arm             │
  │ SD2/SC2 ──► MPU6050 #2  SDA/SCL  [CH2]  →  left_forearm                │
  │ SD3/SC3 ──► MPU6050 #3  SDA/SCL  [CH3]  →  right_forearm               │
  │ SD4/SC4 ──► MPU6050 #4  SDA/SCL  [CH4]  →  left_thigh                  │
  │ SD5/SC5 ──► MPU6050 #5  SDA/SCL  [CH5]  →  right_thigh                 │
  │ SD6/SC6 ──► MPU6050 #6  SDA/SCL  [CH6]  →  left_shin                   │
  │ SD7/SC7 ──► MPU6050 #7  SDA/SCL  [CH7]  →  right_shin                  │
  └──────────────────────────────────────────────────────────────────────────┘

  แต่ละ MPU6050 (GY-521) ต่อเหมือนกันทุกตัว:
    VCC → 3.3V
    GND → GND
    SDA → TCA9548A SDx (ช่องที่ตรงกับตำแหน่งร่างกาย)
    SCL → TCA9548A SCx (ช่องที่ตรงกับตำแหน่งร่างกาย)
    AD0 → GND  (I2C address = 0x68 ทุกตัว — แยกได้เพราะอยู่คนละ TCA channel)
    INT → ไม่ต้องต่อ (ไม่ใช้ interrupt)
    XDA, XCL → ไม่ต้องต่อ (auxiliary I2C ไม่ใช้)

────────────────────────────────────────────────────────────────────────────────
ESP32 DevKit V1 — ขาที่ใช้งาน
────────────────────────────────────────────────────────────────────────────────

  ┌──────────────┬──────────┬──────────────────────────────────────────────┐
  │ Pin Name     │ GPIO     │ ต่อไปที่                                     │
  ├──────────────┼──────────┼──────────────────────────────────────────────┤
  │ 3V3          │ —        │ TCA9548A VCC + RESET + MPU6050 VCC (ทุกตัว) │
  │ GND          │ —        │ TCA9548A GND + A0,A1,A2 + MPU6050 GND ทุกตัว│
  │ GPIO21       │ SDA      │ TCA9548A SDA (I2C master data)               │
  │ GPIO22       │ SCL      │ TCA9548A SCL (I2C master clock)              │
  │ GPIO0        │ BOOT     │ BOOT button (built-in) — ไม่ต้องต่อเพิ่ม    │
  └──────────────┴──────────┴──────────────────────────────────────────────┘

  ⚠ GPIO21 คือ SDA มาตรฐานของ ESP32 (default Wire.begin())
  ⚠ GPIO22 คือ SCL มาตรฐานของ ESP32
  ⚠ GPIO23 จะเป็น fallback SDA ถ้า GPIO21 detect ไม่เจอ TCA9548A

TCA9548A — รายละเอียดขา

  ┌──────────┬──────────────────────────────────────────────────────────────┐
  │ ขา TCA  │ ต่อไปที่                                                     │
  ├──────────┼──────────────────────────────────────────────────────────────┤
  │ VCC      │ ESP32 3V3                                                    │
  │ GND      │ ESP32 GND                                                    │
  │ SDA      │ ESP32 GPIO21  (I2C master bus ← ฝั่ง ESP32)                │
  │ SCL      │ ESP32 GPIO22  (I2C master bus ← ฝั่ง ESP32)                │
  │ RESET    │ 3V3  (active LOW — ต้องดึงสูงเพื่อ enable)                  │
  │ A0       │ GND  ┐                                                       │
  │ A1       │ GND  ├─ กำหนด I2C address = 0x70                           │
  │ A2       │ GND  ┘                                                       │
  │ SD0/SC0  │ MPU6050 #0  (left_upper_arm)   SDA/SCL                      │
  │ SD1/SC1  │ MPU6050 #1  (right_upper_arm)  SDA/SCL                      │
  │ SD2/SC2  │ MPU6050 #2  (left_forearm)     SDA/SCL                      │
  │ SD3/SC3  │ MPU6050 #3  (right_forearm)    SDA/SCL                      │
  │ SD4/SC4  │ MPU6050 #4  (left_thigh)       SDA/SCL                      │
  │ SD5/SC5  │ MPU6050 #5  (right_thigh)      SDA/SCL                      │
  │ SD6/SC6  │ MPU6050 #6  (left_shin)        SDA/SCL                      │
  │ SD7/SC7  │ MPU6050 #7  (right_shin)       SDA/SCL                      │
  └──────────┴──────────────────────────────────────────────────────────────┘

────────────────────────────────────────────────────────────────────────────────
MPU6050 (GY-521) — รายละเอียดขา (เหมือนกันทุกตัว)
────────────────────────────────────────────────────────────────────────────────

  ┌────────────┬──────────────────────────────────────────────────────────────┐
  │ ขา GY-521 │ ต่อไปที่                                                     │
  ├────────────┼──────────────────────────────────────────────────────────────┤
  │ VCC        │ 3.3V  (GY-521 มี regulator ในตัว รองรับ 3.3–5V)            │
  │ GND        │ GND                                                          │
  │ SDA        │ TCA9548A SDx (ช่อง x ที่กำหนดสำหรับ sensor นั้น)           │
  │ SCL        │ TCA9548A SCx (ช่อง x ที่กำหนดสำหรับ sensor นั้น)           │
  │ AD0        │ GND  ← address = 0x68 (ทุกตัวเหมือนกัน OK เพราะแยก channel)│
  │ INT        │ ไม่ต้องต่อ                                                   │
  │ XDA        │ ไม่ต้องต่อ                                                   │
  │ XCL        │ ไม่ต้องต่อ                                                   │
  └────────────┴──────────────────────────────────────────────────────────────┘

────────────────────────────────────────────────────────────────────────────────
Firmware Channel Map (sensor key → TCA channel)
────────────────────────────────────────────────────────────────────────────────

  CH0 = left_upper_arm   (ต้นแขนซ้าย)
  CH1 = right_upper_arm  (ต้นแขนขวา)
  CH2 = left_forearm     (แขนท่อนล่างซ้าย)
  CH3 = right_forearm    (แขนท่อนล่างขวา)
  CH4 = left_thigh       (ต้นขาซ้าย)
  CH5 = right_thigh      (ต้นขาขวา)
  CH6 = left_shin        (หน้าแข้งซ้าย)
  CH7 = right_shin       (หน้าแข้งขวา)

  I2C Addresses:
    TCA9548A    = 0x70
    MPU6050 ×8  = 0x68  (ทุกตัว — แยกกันได้ด้วย TCA channel)

────────────────────────────────────────────────────────────────────────────────
Pull-up Resistors
────────────────────────────────────────────────────────────────────────────────

  - GY-521 บอร์ดส่วนใหญ่มี pull-up 4.7kΩ ที่ SDA/SCL มาแล้ว
  - ฝั่ง ESP32 ↔ TCA9548A: ESP32 มี internal pull-up แต่แนะนำใส่ 4.7kΩ ภายนอก
    ที่ GPIO21 → 3V3 และ GPIO22 → 3V3 เพื่อความเสถียร
  - ฝั่ง TCA9548A ↔ MPU6050: pull-up จาก GY-521 เพียงพอ (ไม่ต้องเพิ่ม)

────────────────────────────────────────────────────────────────────────────────
หมายเหตุ
────────────────────────────────────────────────────────────────────────────────

  - ต่อ GND ร่วมกันทุกอุปกรณ์ (ESP32, TCA9548A, MPU6050 ทั้ง 8)
  - สาย I2C master bus (GPIO21/22) ควรสั้น (< 30 cm) เพื่อลด noise
  - สาย TCA ↔ MPU6050 แต่ละเส้นอิสระ — ยาวได้สูงสุด ~50 cm ต่อ pair
  - ไม่ควรใช้ GPIO23 เป็น MOSI (SPI) และ I2C SDA พร้อมกัน
  - ESP32 3V3 pin จ่ายกระแสได้ ~500 mA; MPU6050 ใช้ ~3.9 mA ต่อตัว
    รวม 8 ตัว = ~31 mA + TCA9548A ~0.5 mA → รวม ~32 mA (OK, ต่ำกว่า 500 mA)
